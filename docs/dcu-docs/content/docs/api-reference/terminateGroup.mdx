The `terminateGroup` endpoint lets the admin claim a `PenaltyState` treasury UTxO — the position a member leaves behind when they exit early. It burns the membership token and releases the locked funds to the admin.

<Callout title="Bound endpoint" type="info">
Obtained from `createDcuSdk(settingsPolicy)`. See [Getting Started](/docs/getting-started).
</Callout>

<Callout title="Admin-gated" type="info">
This endpoint is authorized by the group admin (222) token. The group UTxO is consumed as a **read-only reference input** (not spent) — used only to derive the group policy ID for admin-token authorization; the group validator does not run.
</Callout>

### What it does
- Spends a member's `PenaltyState` treasury UTxO via the `TerminateGroup` redeemer.
- Burns the treasury membership token.
- Releases the locked funds to the admin.

## Configuration

```ts
export type TerminateGroupConfig = {
  /** 28-byte token suffix of the group. */
  groupTokenSuffix: string;

  /** 28-byte account token suffix of the member whose PenaltyState is being claimed. */
  memberAccountTokenSuffix: string;
};
```

## Usage

```ts
import { createDcuSdk } from "@tx-meta/dcu-sdk";

const sdk = createDcuSdk(settingsPolicy);

try {
  const txBuilder = await sdk.terminateGroup(lucid, {
    groupTokenSuffix,
    memberAccountTokenSuffix,
  }).unsafeRun();
  const txHash = await txBuilder.sign.withWallet().complete().then((tx) => tx.submit());
  console.log("Penalty claimed! TxHash:", txHash);
} catch (error) {
  console.error("Failed to terminate:", error);
}
```
