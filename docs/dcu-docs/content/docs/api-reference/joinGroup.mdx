---
title: joinGroup
description: Formally enter a DCU Group and commit to its core logic parameters.
---

The `joinGroup` endpoint allows a user who has minted a User Auth NFT (via `createAccount`) to officially lock their commitment into a specific Decentralized Credit Union.

### Context within the DCU Protocol
In a standard Rotating Savings and Credit Association (ROSCA), groups have a defined capacity of members who mutually agree to pool funds. By calling `joinGroup`, the user's wallet is cryptographically bound to the rules set by the Group Administrator during `createGroup` (such as the `contribution_fee`, `interval_length`, and `num_intervals`).

## Configuration

To join a group, a user must provide proof of their identity and prove the existence of the group.

```ts
import { UTxO } from "@lucid-evolution/lucid";

export type JoinGroupConfig = {
    /** The master Group UTxO containing the rules and current member count */
    groupUtxo: UTxO;
    
    /** The Auth UTxO representing your identity (created via createAccount) */
    accountUtxo: UTxO;
    
    /** 
     * The Admin Auth UTxO required to securely update the group's 
     * state by incrementing the `member_count` on-chain.
     */
    adminUtxo: UTxO;
    
    /** The exact lovelace amount locked in the Treasury as your cyclic fee */
    contributionAmount: bigint;
};
```

## What Happens On-Chain

The SDK builds a single atomic transaction that:
1. **Spends** the `groupUtxo` and returns it with an updated datum. The `member_count` is incremented and the next available rotation slot is assigned to this member.
2. **Mints** 1 Treasury Membership NFT under the Treasury policy, uniquely identified by the member's Account token name.
3. **Locks** the `contributionAmount` (lovelace) together with the new membership token into the Treasury Validator. The datum is pre-populated with the full `contribution_list` vesting schedule for all intervals.
4. **Returns** the `accountUtxo` unmodified to the user's wallet. The identity credential is not consumed.

## Example Execution

```ts
import { joinGroup } from "@dcu/sdk";

try {
  const txBuilder = await joinGroup(lucid, {
    groupUtxo,
    accountUtxo,
    adminUtxo,
    contributionAmount: 50_000_000n // 50 ADA
  }).unsafeRun();

  const txHash = await txBuilder.sign.withWallet().complete().then(tx => tx.submit());
  console.log("Joined group:", txHash);
} catch (error) {
  console.error("Failed to join:", error);
}
```
