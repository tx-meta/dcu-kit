The `createGroup` endpoint establishes a new group, defining the rules, fees, and collateral/payout policy for a specific ROSCA pod.

<Callout title="Bound endpoint" type="info">
`createGroup` is obtained from `createDcuSdk(settingsPolicy)`, not imported standalone. See [Getting Started](/docs/getting-started) for the one-time settings setup.
</Callout>

<Callout title="CIP-68 token pair" type="info">
`createGroup` mints a CIP-68 pair: a reference (100) token locked at the Group Validator holding the group datum, and an admin (222) token sent to your wallet as the admin credential.
</Callout>

### Core Responsibilities
When executed, the SDK constructs a transaction that:
1. Mints the group reference + admin token pair under the group policy ID.
2. Locks the `groupDatum` rules as an inline datum at the Group Validator, with the `creator_bond` ADA.
3. Spends `utxoToSpend` to fund the transaction and supply token-name entropy.

## Configuration

```ts
import { OutRef } from "@lucid-evolution/lucid";
import { GroupDatum } from "@tx-meta/dcu-kit";

export type CreateGroupConfig = {
  /** Display name shown by wallets — written into metadata["name"]. */
  groupName: string;

  /** The core ruleset of the new group. */
  groupDatum: GroupDatum;

  /** Reference to the wallet UTxO spent to fund creation and seed token names. */
  utxoToSpend: OutRef;
};
```

### GroupDatum fields

All numeric fields are `bigint`; time values are POSIX milliseconds. Several fields are **constrained at creation** by the validator and are set to their initial values by later transitions.

| Field | Type | Description |
|-------|------|-------------|
| `contribution_fee` | `bigint` | Amount each member contributes per round (contribution asset) |
| `contribution_fee_policyid` / `_assetname` | `string` | Contribution asset. `""`/`""` for ADA |
| `joining_fee` (+ `_policyid`/`_assetname`) | `bigint`/`string` | One-time fee paid to the admin at join |
| `penalty_fee` (+ `_policyid`/`_assetname`) | `bigint`/`string` | Charged on early exit |
| `grace_period_length` | `bigint` | Grace window granted to a defaulting member (ms) |
| `creator_bond` | `bigint` | ADA locked at creation, returned on `deleteGroup`. Deters spam |
| `interval_length` | `bigint` | Duration of each round (ms) |
| `num_rounds` | `bigint` | **Must be `0n` at creation.** Set to `member_count` by `startGroup` |
| `max_members` | `bigint` | Group capacity |
| `collateral_rounds` | `bigint` | Join floor in rounds of fee: `1n` = PerRound (default), `max_members` = FullUpfront, `k` = partial |
| `payout_mode` | `"Push" \| "Pull"` | `Push` pays the borrower's wallet; `Pull` earmarks into their treasury |
| `member_count` | `bigint` | **`0n` at creation** |
| `is_active` | `boolean` | `true` at creation |
| `is_started` | `boolean` | **`false` at creation.** Set by `startGroup` |
| `start_time` | `bigint` | **Must be `0n` at creation.** Set to the tx lower bound by `startGroup` |
| `last_distributed_round` | `bigint` | **`-1n` at creation** |
| `creator_payment_credential` | `string` | Admin payment key hash |
| `member_token_names` | `string[]` | **`[]` at creation.** Appended on each join |

**Example:**

```ts
import { GroupDatum } from "@tx-meta/dcu-kit";

const groupDatum: GroupDatum = {
  contribution_fee:           5_000_000n,      // 5 ADA per round
  contribution_fee_policyid:  "",              // ADA
  contribution_fee_assetname: "",
  joining_fee:                2_000_000n,
  joining_fee_policyid:       "",
  joining_fee_assetname:      "",
  penalty_fee:                1_000_000n,
  penalty_fee_policyid:       "",
  penalty_fee_assetname:      "",
  grace_period_length:        0n,
  creator_bond:               5_000_000n,      // >= contribution_fee recommended
  interval_length:            2_592_000_000n,  // 30 days in ms
  num_rounds:                 0n,              // MUST be 0 — startGroup sets it
  max_members:                5n,              // capacity
  collateral_rounds:          1n,              // PerRound (top up via contribute)
  payout_mode:                "Push",
  member_count:               0n,
  is_active:                  true,
  is_started:                 false,
  start_time:                 0n,              // MUST be 0 — startGroup anchors it
  last_distributed_round:     -1n,
  creator_payment_credential: adminPkh,        // admin payment key hash
  member_token_names:         [],
};
```

## Usage

```ts
import { createDcuSdk } from "@tx-meta/dcu-kit";

const sdk = createDcuSdk(settingsPolicy);

try {
  const txBuilder = await sdk.createGroup(lucid, {
    groupName: "Family Chama",
    groupDatum,
    utxoToSpend,
  }).unsafeRun();
  const txHash = await txBuilder.sign.withWallet().complete().then((tx) => tx.submit());
  console.log("Group created!", txHash);
} catch (error) {
  console.error("Failed to create group:", error);
}
```
