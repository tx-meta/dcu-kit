---
title: exitGroup
description: Safely process an early or mature exit from a DCU protocol pod.

The `exitGroup` endpoint handles one of the most critical flows in a Rotating Savings and Credit Association: Membership departure.

Depending on when a user chooses to exit relative to the `start_time` and maturity interval of the Group, the smart contracts enforce completely different financial branching logic.

<Callout title="Bound endpoint" type="info">
Obtained from `createDcuSdk(settingsPolicy)`. See [Getting Started](/docs/getting-started).
</Callout>

## Configuration

```ts
import { UTxO } from "@lucid-evolution/lucid";

export type ExitGroupConfig = {
    /** 28-byte token suffix of the group to exit. */
    groupTokenSuffix: string;

    /**
     * Optional 28-byte account token suffix used when joining. If omitted, the
     * endpoint auto-detects it by scanning the wallet's account tokens against
     * treasury UTxOs (handles wallets holding multiple account tokens).
     */
    accountTokenSuffix?: string;

    /** POSIX ms. Use the emulator clock for tests; omit on live networks. */
    currentTime?: bigint;

    /** Reference scripts (from deploy-scripts) to keep the tx under the 16 KB limit. */
    scriptRefs?: { treasury?: UTxO; group?: UTxO };
};
```

## Protocol Branching Logic

The **maturity threshold** is the point at which the group has completed all its rotation cycles:

```
maturity_threshold = start_time + (num_rounds × interval_length)
```

Both values come directly from the group's `GroupDatum`.

### 1. Early Exit (Penalty State)
If the absolute transaction time (enforced by the Cardano slot) is earlier than the Group's maturity threshold, the SDK handles this as an **Early Exit**.
* The Treasury UTxO transitions to a locked `PenaltyState`. The membership token is **not** burned; it remains locked inside the Treasury UTxO.
* The global Group `member_count` decrements.
* A **`penalty_fee`** is permanently deducted from the locked balance.

### 2. Mature Exit (Graceful Refund)
If the Group's rotation has successfully run its course and elapsed past the maturity threshold, this is handled as a **Mature Exit**.
* The membership token is burned.
* The global Group `member_count` decrements.
* The total remaining balance is refunded to the exiting member's wallet.

## Usage

```ts
import { createDcuSdk } from "@tx-meta/dcu-kit";

const sdk = createDcuSdk(settingsPolicy);

try {
  const txBuilder = await sdk.exitGroup(lucid, { groupTokenSuffix }).unsafeRun();

  const txHash = await txBuilder.sign.withWallet().complete().then((tx) => tx.submit());
  console.log("Exited group successfully! TxHash:", txHash);
} catch (error) {
  console.error("Exit failed:", error);
}
```
